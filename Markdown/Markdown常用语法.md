# Markdown常用语法

## 常用的

```markdown
1.上标
    爆米<sup>TM</sup>
2.下标
    H<sub>2</sub>O  CO<sub>2</sub>
3.键盘文字
    <kbd>...</kbd>
4. 目录
	[toc]  自动更新
5. 高亮文字
	==文字==
6. 文本居中
	<center>这是要居中的内容</center>
7. 不能正常显示的符号，可以使用反斜杠\ 来帮助显示
8. 注释 
<!-- 注释 -->
9. 表格内，可以使用 <Br/> 来帮助换行
```

## 不常用的

    1. 根号
    	√<span style="text-decoration:overline">x</span>
